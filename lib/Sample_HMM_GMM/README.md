## Installation
```bash
# Step 1 使用 git 下載專案
git clone https://github.com/Amiannn/Sample-HMM-GMM.git
cd Sample-HMM-GMM

# Step 2 使用 Miniconda 建立虛擬 python 環境
conda create --name hmm python=3.7
conda activate hmm

# Step 3 安裝套件
pip3 install -r requirements.txt
```
